# Awesome AI Security 贡献指南

## 贡献方式
- 添加或更新资源，并附上简短、中立的说明和链接。
- 修正文档问题（失效链接、措辞、格式）。
- 报告问题时请清楚说明问题内容和出现位置。

## Pull Request
1. Fork → 创建分支 → 提交更改。
2. 发起 PR，说明更改内容和原因。
3. 关联相关 issue。

## 风格
- 仅使用 Markdown；每个条目一行：`[名称](https://link) — 简短描述。`
- 保持中立语气，优先使用官方/规范链接。
- 避免重复，选择最合适的章节。

## 许可证
贡献即表示你同意 **CC0 1.0**。
